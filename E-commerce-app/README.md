
## To Install and run the application
npx create-vite E-commerce-app --template react-ts,
cd E-commerce-app,
npm install @reduxjs/toolkit,
npm install react-redux,
npm install @tanstack/react-query,
npm install axios,
npm run dev,

Brwose: http://localhost:5173
